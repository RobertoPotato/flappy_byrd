# Flappy Byrd 

- A Python Implementation of the Flappy Bird game. Added a 
dive method for when things get tight. 
- Controls are the 
Up and Down arrow Keys on the keyboard.
  
# Built With 
- Python 3.8.5
- Turtle Graphics
- Love
